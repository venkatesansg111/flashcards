What is Azure Kubernetes Service (AKS)?
AKS is a managed container orchestration service provided by Microsoft Azure that simplifies deploying, managing, and operating Kubernetes.
What is Kubernetes?
Kubernetes is an open-source platform for automating deployment, scaling, and operations of application containers across clusters of hosts.
What is a Deployment in Kubernetes?
A Deployment provides declarative updates to applications, managing the deployment of Pods and ReplicaSets.
What is a Service in Kubernetes?
A Service is an abstraction that defines a logical set of Pods and a policy to access them, typically used to expose applications running in a cluster.
What is Helm?
Helm is a package manager for Kubernetes that helps to define, install, and upgrade complex Kubernetes applications.
What is the Azure Container Registry (ACR)?
ACR is a managed Docker registry service based on the open-source Docker Registry 2.0, used for storing and managing container images.
What is role-based access control (RBAC) in AKS?
RBAC is a mechanism for defining and enforcing permissions for users and applications to access resources within an AKS cluster.
What is the Kubernetes API Server?
The Kubernetes API Server is a component of the Kubernetes control plane that exposes the Kubernetes API.
What is a DaemonSet in Kubernetes?
A DaemonSet ensures that all (or some) Nodes run a copy of a Pod, commonly used for log collection or monitoring.
What are Kubernetes Ingress Controllers?
Ingress Controllers manage external access to services in a Kubernetes cluster, typically via HTTP/HTTPS.
What is the Azure CLI?
Azure CLI is a set of commands used to create and manage Azure resources, including AKS clusters, from the command line.
What is a StatefulSet in Kubernetes?
A StatefulSet is a workload API object used to manage stateful applications, ensuring that each Pod has a unique identity and stable storage.
What is the Kubernetes Scheduler?
The Kubernetes Scheduler is responsible for placing Pods on Nodes based on resource availability and other constraints.
What is Horizontal Pod Autoscaler (HPA)?
HPA automatically scales the number of Pods in a deployment based on observed CPU utilization or other select metrics.
What is a ConfigMap in Kubernetes?
A ConfigMap is a Kubernetes API object that stores configuration data in key-value pairs, allowing you to decouple configuration artifacts from image content.
